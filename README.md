# WFR_Florida-Plant-Atlas
Materials for USF Libraries/USF Herbarium Workshop for Researchers on the Atlas of Florida Plants and data exploration with the R statistical computing environment.

This repository contains the files and code necessary to follow along with a workshop presented on April 7th 2026.

# Outline

What's the question?
- Biodiversity
- Invasive species
- Establishing collecting priorities

Data retrieval and cleanup
- Species-focused
- People-focused
- Locality-focused
- County-focused

Quick R applications
- Fast mapping species distributions (by county and location)
- Species-sampling curves
